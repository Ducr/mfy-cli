### 初始化配置

* 配置入口
```jsx static
  #!/usr/bin/env node  
```
* npm link 到全局目录

* 配置基本命令 在 './bin/mfy' 目录下面

## 使用开发配置
### 创建项目
```
mfy-cli create [project-name]
```
### config 配置项目
#### default template
```
https://github.com/mfy-template
```
#### 修改自定义的项目源

```
mfy-cli config -sm
```
### 修改成默认源
```
mfy-cli config -smc
```
